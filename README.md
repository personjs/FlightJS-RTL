# :airplane: FlightJS RTL

An all-in-one solution for displaying ADS-B data.

## 📦 Tech Stack

- [Svelte 5.x](https://svelte.dev/blog/svelte-5)
- [Vite](https://vitejs.dev/) - Next-gen frontend tooling
- [TypeScript](https://www.typescriptlang.org/) (optional)
- [Leaflet](https://leafletjs.com/)
- [Dump1090](https://github.com/antirez/dump1090) - Mode S decoder for RTL-SDR devices.

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher)
- [pnpm](https://pnpm.io/) or `npm` / `yarn`

### Install dependencies

```bash
pnpm install
# or
npm install
```
